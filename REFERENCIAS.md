# Referencias

## Tecnologías utilizadas

- [Express.js](https://expressjs.com/)  
  Framework de Node.js para crear y gestionar rutas HTTP.

- [Zod](https://zod.dev/)  
  Librería de validación y parsing de esquemas para JavaScript/TypeScript.

- [node:sqlite](https://nodejs.org/api/sqlite.html)  
  Módulo nativo de Node.js 22 para trabajar con bases de datos SQLite sin dependencias externas.

- [dotenv (--env-file)](https://nodejs.org/en/learn/command-line/how-to-read-environment-variables-from-nodejs)  
  Carga de variables de entorno usando la bandera nativa `--env-file` de Node.js 22.
